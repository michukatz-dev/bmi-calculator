
# BMI Calculator
Interactive web application that calculates Body Mass Index (BMI) based on user input.

## Features
- Height conversion (cm to meters)
- BMI classification logic
- Real-time DOM updates
- Input validation

## Tech Stack

- CSS
- JavaScript
